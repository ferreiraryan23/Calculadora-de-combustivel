🚀 Como executar
Certifique-se de ter o Python 3 instalado em sua máquina.

🛠️ Regras do Código
Validação: O programa verifica se os valores digitados são maiores que zero.

Precisão: Os resultados de litros consumidos e custo total são exibidos com duas casas decimais.
